# 🔐 Password Generator (Python)

This project is a simple Python-based Password Generator that creates secure random passwords based on user-defined length.

## 🚀 Features
- Generates random passwords
- User decides password length
- Uses letters (A–Z, a–z) and digits (0–9)
- Simple and fast command-line program

## 🛠️ Technologies Used
- Python
- random module
- string module
- loops and conditionals

## 📌 How It Works
1. User enters desired password length
2. Program selects random characters from letters and digits
3. Characters are combined to form a password
4. Final password is displayed

## ▶️ Example
Enter the desired password Length: 6  
Generated password is: aB3k9Z

## 🎯 Learning Outcome
This project helps understand:
- Random selection in Python
- String manipulation
- Looping concepts
- Basic security logic

Beginner-level Python mini project for logic building and practice
